# JavaScript Practice Repository

This repository contains a variety of JavaScript projects, exercises, and examples aimed at improving coding skills and understanding of JavaScript fundamentals.

## Features
- **Interactive Games**: Guessing Game, Rock-Paper-Scissors, and more.
- **Array & Number Operations**: Average calculation, sorting, and filtering exercises.
- **DOM Manipulation**: Projects that demonstrate updating the web page dynamically.
- **Problem Solving**: Small coding challenges to strengthen logic and algorithm skills.

## Purpose
This repository is designed for beginners and intermediate developers to practice JavaScript in a structured and fun way. All projects are self-contained and easy to understand.

## Usage
- Clone the repository
- Open the HTML files in a browser (for browser-based projects)
- Explore JavaScript logic in the console or using the scripts

Happy coding! 🚀
